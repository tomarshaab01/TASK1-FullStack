# TASK1 FullStack

A simple e-commerce website made with HTML, CSS and JavaScript.

## Pages

- `index.html` - Product listing page (Task 1)
- `product.html` - Product detail page, opens when you click a product (Task 2)
- `cart.html` - Cart page with bill summary (Task 3)

## APIs Used

- All products: https://dummyjson.com/products?limit=194
- Single product: https://dummyjson.com/products/{id}

## How to Use

1. Open `index.html` in your browser
2. Search or filter products
3. Click any product card to see its details
4. Add to cart and go to cart page to see the bill
